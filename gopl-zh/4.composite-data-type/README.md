数组是一个由固定长度的特定类型元素组成的序列

```
var a [3]int
var q [3]int = [3]int{1, 2, 3}
var r [3]int = [3]int{1, 2}
q := [...]int{1, 2, 3}
r := [...]int{99: -1}
```

数组的长度是数组类型的一个组成部分，因此[3]int和[4]int是两种不同的数组类型
